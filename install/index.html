Installation

First of all clone the repository from ISP master branch. Just open your terminal and navigate whenver you want to place ISP and type in the terminal:

`git clone https://github.com/ProjectISP/ISP.git`

Then, go to the folder ./ISP and to change to the ongoing version ISP 2.0 type in your terminal 

`git checkout ISP_2.0_development`

To be updated, from ./ISP type in your terminal, 

`git pull`

The installation file will install the requirements and will compile the source packages. At the end of the installation the user will be asked for an alias "isp". It is needed to have previously installed anaconda. Just in case the alias isp do not open the software, go to ./ISP and type:

`condo activate isp` or `source activate isp`
`python start_isp.py`

Installation for end users (Mac and Linux)

To run the installation go to ./isp/installation and type ./ISP_installer.sh

Warning: We have found that in some Mac computers is difficult to compile the third package NLL, so for this reason inside the folder ./ISP/isp/mac_bin/bin, you will find NLL fully compiled. Then, you can move this files from ./ISP/isp/mac_bin/bin to ./ISP/isp/NLL7/bin that is the folder where ISP will look for running this program.

Installation for developers (Mac and Linux)

To run the installation go to ./isp/installation and type ./ISP_installer_search.sh

Installation for Windows users

1.- Install WSL (Windows Subsystem for Linux) (instalar version 2) y Ubuntu 20.04: - https://docs.microsoft.com/en-us/windows/wsl/install-win10 2.- Install VcXsrv Windows X Server: - https://sourceforge.net/p/vcxsrv/wiki/Using%20VcXsrv%20Windows%20X%20Server/ - Start with option -nowgl, and on Ubuntu "export LIBGL_ALWAYS_INDIRECT=" (https://superuser.com/questions/1502708/wsl-qt-application-fails-to-start) inside Ubuntu: export Display=:0

May be point 2 fails, so you need to try: - Start in " "C:\Program Files\VcXsrv\vcxsrv.exe" :0 -multiwindow -clipboard -wgl -ac ", nest type in the WSL terminal: export DISPLAY_NUMBER="0.0" export DISPLAY=$(grep -m 1 nameserver /etc/resolv.conf | awk '{print $2}'):$DISPLAY_NUMBER See: https://stackoverflow.com/questions/61860208/wsl-2-run-graphical-linux-desktop-applications-from-windows-10-bash-shell-erro

3.- Install Anaconda: - https://linuxize.com/post/how-to-install-anaconda-on-ubuntu-20-04/ - Add to path "export PATH=~/anaconda3/bin:$PATH" al fichero .bashrc

4.- Install normally ISP:

go to ./isp/installation and type ./ISP_installer.sh

Uninstall ISP

Firs remove the folder ISP, then type in your terminal:

`conda remove --name isp --all`
